== Description ==

Плагин для быстрой и удобной установки виджета "Уведомление о cookies" от сервиса "WIDG.net".
[WIDG.net](https://widg.net) - веб-сервис для развития вашего бизнеса и повышения продаж.

**Уведомление о файлах cookie** предоставляет простой настраиваемый баннер веб-сайта, который можно использовать, чтобы помочь вашему веб-сайту соответствовать требованиям о согласии на использование файлов cookie в соответствии с законом о персональных данных в РФ(№ 152-ФЗ).

== Frequently Asked Questions ==

= Что такое согласие на использование файлов cookie? =

Это уведомление об использовании cookie файлов на вашем сайте через всплывающее окно, баннер или уведомление. Он позволяет вам написать свое сообщение, добавить ссылку на политику конфиденциальности и добавить кнопки для получения согласия от пользователей.

= Как добавить Cookie виджет на сайт? =

Вам нужно настроить оформление виджета на сайте widg.net. Сохраните изменения и скопируйте код виджета. Перейдите к настройке плагина на своем сайте, вставте код виджета и сохраните настройку.

= У вас есть помощь при установке виджета? =

Да, мы оказываем бесплатную помощь с виджетами и их интеграцией на сайт для всех клиентов. Не стесняйтесь обращаться в нашу службу поддержки.

= Есть ли ограничения у пробной версии? =

Тестовая версия виджета ничем не отличается от рабочей. Все функции включены.

= У вас бывают дополнительные скидки? =

Да, периодически мы проводим акции, и вы можете получить специальный купон. Размер скидок по купонам до 50%.

== Installation ==

Процесс инсталляции плагина стандартен для WordPress.

1. Найти плагин в панели администратора вашего сайта:
Плагины->Добавить новый->Поле ввода "Поиск плагинов"-> Ввести "widgnet_1020".
2. Установить найденный плагин.
3. Активировать плагин:
Плагины->Установленные->Активировать.
4. Перейти к настройке плагина->Вставить код виджета с сайта widg.net->Сохранить настройку
